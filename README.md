# GraphQL-TypeScript-Example
A Basic example of using Graphql with Typescript and Express-GraphQL

<h1>Start Backend</h1>
1.npm install
<br></br>
2.npm start


<h1>Queries</h1>
users - Fetches all Users
user(id:string) - fetches a specific user based on a ID.


<h1>Mutations</h1>
createUser - Creates a user
